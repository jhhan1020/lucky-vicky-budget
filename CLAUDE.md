# 이 프로젝트에 대해

개발 연습용 폴더입니다.

## 기술 스택

- **Next.js** — 웹사이트를 만드는 프레임워크 (React 기반)
- **Supabase** — 데이터베이스 및 로그인 기능을 제공하는 서비스
- **Vercel** — 만든 웹사이트를 인터넷에 올려주는 배포 서비스

## 소통 방식

- 개발 용어는 비개발자도 이해할 수 있게 쉬운 말로 설명한다
- 지금 무엇을 하고 있는지 자주 알려준다 (예: "이제 버튼을 추가하고 있어요", "데이터베이스에 연결 중이에요")

## 작업 방식

- 사용자가 지시한 것만 단계적으로 수행한다 — 요청하지 않은 기능이나 리팩토링은 하지 않는다
- 컴퓨터가 할 수 있는 건 최대한 직접 처리한다
- 사람이 직접 해야 하는 것(Vercel 배포, Supabase 설정 등)은 쉬운 말로 단계별 안내를 해준다
- 작업이 끝나면 항상 결과를 바로 확인할 수 있도록 미리보기를 열어준다
